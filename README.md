# 📂 Class Repository

This repository is for uploading your implementations of research papers.
Each group (or individual) should create their own folder and upload their work properly.

---

## 📁 What you need to do

* Create a folder with your group name
  (e.g., `group_01`, `group_rl_team`, or your names)

* Inside your folder, include:

  * Your code
  * The research paper (PDF)
  * A `README.md` file

---

## 📝 What should be in your README

Keep it simple and clear:

* Names of group members
* Paper title (and optionally authors)
* A short description of what you implemented
* Anything interesting (results, observations, issues, etc.)

---

## ✍️ Add your entry below

After creating your folder, add a short paragraph like this:

**example_group_01** – Alice, Bob
Implemented *Paper Title*. This project focuses on briefly explaining what you did and what your implementation achieves.

**example_group_02** – John, Priya
Worked on *Another Paper*. We implemented the core idea and tested it on a small example dataset.

(Just keep it short and readable — 2–3 lines is enough.)
